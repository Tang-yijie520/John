Getting Started

Welcome to Simply Rugby Club Management System. This guide will help you use the main features of the application.

Login

  1. Launch the Simply Rugby Club application.
  2. Enter your username and password.
  3. Click the Login button.
  4. If successful, you will be taken to the main menu.

If you enter an invalid username or password, a warning message will be displayed.

Managing Members

Club members (players) can be added, edited, and deleted from the Members window.

To add a new member: Click Members → Add New, fill in the details, then click Save.
To edit a member: Select a member in the list, click Edit, update the details, and click Save.
To delete a member: Select a member and click Delete. A confirmation message will appear.
To search: Type a name or SRU number in the search box.

Managing Squads

Squads organise players into teams for a particular season and age grade.

Create a squad: Enter squad name, select age grade, enter season, then click Create.
Add a player to a squad: Select a squad and an available player, then click Add →.
Remove a player: Select a player in the squad and click ← Remove.
Delete a squad: Select the squad and click Delete Selected. Players are automatically unassigned.

Recording Matches

  1. Open the Matches window from the main menu.
  2. Select the squad that played the match.
  3. Enter the date, opponent, venue, home score and away score.
  4. Click Save Match.

Match results (Win, Loss or Draw) are calculated automatically. Select a match and click Delete Selected Match to remove it.

Skill Assessment

Coaches can record skill assessments for each player on a 1–5 scale.

  1. Open the Skill Assessment window.
  2. Select a player and the assessment date.
  3. Use the sliders to score Passing, Tackling, Kicking, Running with Ball, and Teamwork.
  4. Add optional comments and click Save Assessment.

The player's training profile averages will be updated automatically.

Training Profile

The Training Profile window shows a summary of a player's progress.

  • Select a player from the dropdown.
  • View their average skill scores and matches played.
  • The assessment history is displayed in a table.

User Management (Admin only)

Administrators can create new user accounts, reset passwords, and deactivate users.

Create a user: Enter username, password, email, role, and click Create.
Reset password: Select a user, type a new password, then click Reset Password.
Deactivate a user: Select the user and click Deactivate User.

Frequently Asked Questions

Q: I forgot my password. What should I do?
A: Ask the administrator to reset your password from the User Management window.

Q: Why can't I access User Management?
A: Only users with the Admin role can open the User Management window.

Q: Can I delete a match by mistake?
A: The system will always ask for confirmation before deleting a record.

Q: Where is the data stored?
A: All data is stored locally in a SQLite database file (simplyrugby.db).
